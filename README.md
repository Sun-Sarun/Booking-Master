# TripBookingSystem
## Set Up System
### Step 1
Install [Xampp](https://www.apachefriends.org/download.html)\
after install Xampp, move to Xampp dirctory's /lampp/htdoc/ the clone the project.
### Step 2
Execute Xampp then start apache and mysql servers.
### Step 3
Open you favor Broswer to create database -> [create-mysql-database](http://localhost/phpmyadmin/index.php?route=/table/create&db=tripBookingPOS)\
### Step 4
Import database and data structure ->[mysql-import-to-database](http://localhost/phpmyadmin/index.php?route=/database/import&db=tripBookingPOS)\
then select file: /lampp/htdocs/tripBookingPOS/database/data/tripBookingPOS.sql
### Step 5
Open [localhost](https://localhost/tripBookingPOS/home/index.php)
